# DeCodeX Project Changelog

## Version 2.1.0 (Current)
- Added DNA → mRNA transcription visualization mode
- Improved base ring rendering with better visual hierarchy
- Added support for both strands visualization (Coding/Non-Template)
- Enhanced H-bond visualization with variable bond counts (2 for A-T, 3 for G-C)

## Version 2.0.0
- Complete redesign with modern dark theme
- Canvas-based rendering for smooth 60fps animation
- Interactive helix rotation with drag support
- Dynamic base pair ring structures (purine/pyrimidine)
- Responsive design for all screen sizes
- Accessibility improvements (ARIA labels)

## Version 1.0.0
- Initial release
- Basic helix visualization
- Static information panel
